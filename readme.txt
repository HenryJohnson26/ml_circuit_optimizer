File structure
Root
|   data
    | optimized     #Circuits optimized by qiskit generated by optimize_circuits.py
    | raw           #Raw random circuits generated by gen_circuits.py
    | tokenized     #Tokenized raw and optimized circuits with circuit info in .json format generated by tokenize_circuits.py
|   scripts
    | gen_circuits.py   #Generates random circuits for data
    | optimize_circuits.py         #optimizes circuits using qiskit qasm
    | tokenize_circuits.py         #tokenizes circuits to be read for model generation
| src
    | __init__.py #creates a package to be used in scripts
    | circuit_gen.py #generates random circuits
    | cnn.py         #read tokenized circuits, define loss functions, train model
    | qiskit_passes.py #optimize circuits
    | tokenizers.py #define tokens and tokenize circuits
.gitignore
quantum_optimizer_model.pth #store trained model
readme.txt
report.txt
requirements.txt #name required packages to be installed in order to run


How to run:
start a python environment using either python venv or conda by running

      python -m venv .path_to_env_folder
      .path_to_env_folder/Scripts/activate #windows powershell
      source .path_to_env_folder/bin/activate #linux

or

    conda create --name <env-name>
    conda activate <environment name>

then install required packages with pip, or whatever you use
     pip install -r requirements.txt
You may have to create a temporary folder to install all of the packages since they are large.
    TMPDIR=/path/to/temp/folder pip install -r requirements.txt

You can generate data with
*from root directory*
      python -m scripts.gen_circuits
      python -m scripts.optimize_circuits
      python -m scripts.tokenize_circuits

then you can make and test a model with
*from root directory*
      python -m src.cnn
